[ 회원가입 기능 구현 ]
$jsp
URI:/signup 에 정보를 입력하면 회원의 정보가 DB에 저장되는 기능

@Controller
서비스에게 화면에 입력받은 정보를 넘겨주면 
ID가 중복인지 확인하고 중복이 아니면 데이터베이스에 회원정보를 저장시키는 기능을 전달

@Service
화면에서 입력받은 정보를 DAO를 통해 데이터베이스에 저장 시킨다
함수 구현시 >
DB에 저장된 값을 돌려받을때 null이어서는 안된다.
입력받은 모든 정보는 null이어서는 안된다
id가 중복되지 않으면 데이터베이스에 값을 저장하고 회원가입 처리한다.

@DAO

데이터베이스에서 중복된 아이디인지 확인한 id를 가지고 온다.
#mapper
select id from member
insert into member values(속성명들)
insert 문을 사용해서 회원 정보를 DB에 입력한다.

@VO
memeber 테이블에 저장되어 있는 속성값들을 저장한 클래스
 */
