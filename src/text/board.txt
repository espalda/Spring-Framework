[컨트롤러] @Controller @Autowired @RequestMapping
[서비스] @Service @Autowired
[DAO] - Mapper연결

@Controller
서비스에게 DB에 저장된 게시글 리스트 정보들를 가져와서 화면에 보여주는 기능을 전달
Controller에서 DB 정보를 담아올 객체를 먼저 선언, VO생성

@Service
DAO에게서 DB에 저장된 리스트 목록을 가져오도록 요청

@DAO
Mapper를 통해서 DB 에 저장되어 있는 속성들 전부를 가져옴
 - Mapper
select * from board
@VO
DB에 저장된 속성들을 편하게 가져오기 위해서 생성

암기
c:forEach var="board" items="${list}"
${board.title}

@Controller

@Service
게시글 번호가 입력받은 게시글 번호와 같다면
데이터베이스 board 정보를 가져오게
다오에게 일을 시킨다. 

@DAO 
mapper를 통해서 데이터베이스 board 정보를 가져온다.

- mapper
매개변수 num과 다오를 통해 가져온 데이터데이스의 num가 같으면 DAO를 통해 데이터베이스 값을 가져온다.
게시글 번호가 같으면 내용을 가져온다.
